README

This project is a modularized code library for the
game engine Godot created by Matthew Fernandez and 
Ryan English

This project includes functionality for a 3d first person controller, 3d third person controller, a UI, an inventory
system, character/npc state, menu/level system, use system, dialog system, spawning objects/enemies, doors and object interactivity, 2d side scroller controller and a 2d top down controller
